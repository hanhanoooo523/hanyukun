当 github.io 无法访问时，可用下面方式让任何人用浏览器直接看演示稿（无需登录 GitHub）。配图已内嵌在演示 HTML 内，单文件即可展示。

通过 HTML Preview 读取仓库里的原始文件，不依赖 GitHub Pages。

方式二：最稳 — 单文件发给别人

  • 从仓库下载 StyleShift-Presentation.html 或 StyleShift-设计演示.html（或发邮件 / 网盘）。
  • 对方用 Chrome / Edge 等直接双击打开该文件即可（无需服务器）。
