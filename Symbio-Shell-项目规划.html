Symbio-Shell 项目规划

思辨设计 / 虚构装置路线——不制作实物，不涉及台架或生物实验；以叙事、视觉与表述规范为主。

在纸面与屏幕上把体系说圆：系统图、分镜、网站或幻灯可达答辩/作品集程度；对外明确「概念提案」即可。

文档性质：概念/构思阶段专用；不包含可握持样机、工作原型或实验室验证。Markdown 同源：Symbio-Shell-项目规划.md

1 项目概述

项目说明
名称Symbio-Shell（共生壳）
一句话定义基于生理反馈与合成生物代谢逻辑的活性手机界面（概念壳体）。
核心价值将「数字焦虑、废热、依赖」转译为可感知的多通道反馈，推动从消耗到轮回的材料与伦理叙事。
规划边界纯概念与可观看的叙事资产（图文、网站、幻灯、短片稿等）。不做实物、不做生物/化学实测。对外可称思辨设计 / 概念提案。

成功标准（建议同时满足其中 3 条即可视为阶段成功）

  1. 叙事与技术路径自洽：四层结构、代谢闭环、行为干预逻辑可用一页「系统图 + 时序」讲清（可标「推测」「未验证」）。
  2. （可选） 配 2–3 条公开文献或常识级脚注，说明机制「对应哪类讨论」即可——不要求自做实验或数据。
  3. 视觉与叙事资产可展示：解剖图、分镜、展板/KV、网页或短片稿（含 AI 成图）至少一类达答辩/作品集完成度。
  4. 伦理与表述备忘录 + 明确构思性质；附简短「我们不承诺什么」。

2 阶段总览（推荐节奏）

以下为 约 4–6 个月 的轻量切分；若仅为单次答辩，可用 6–8 周冲刺（见表 2.1）。

阶段周期（参考）目标
P0 对齐与范围第 1–2 周锁定叙事、成功标准、明确无实物、成果形式（展赛 / 作品集 / 网页）。
P1 设定与依据第 2–6 周系统框图、假设表、自愿的文献笔记（无需实验）、名词与「推测」标注规范。
P2 叙事与视觉与 P1 并行主视觉与解剖图迭代、分镜、概念页与介绍页统一、可选动效/短片脚本。
P3 整合与预演第 5–8 周答辩 deck、讲稿、模拟追问；汇总为可分享入口（静态站或 PDF）。
P4 复盘与延伸持续若未来有人愿做实物，另立子项目；此处只归档叙事与素材说明。

2.1 答辩「压缩版」里程碑（示例 · 无实物）

周次交付
W1–W2一页系统图 + 假设表 + 对外一句「本项目为概念提案」
W3–W4叙事定稿 + 主视觉/KV + 2 分钟口播稿
W5–W6分镜深化或 30–60 秒影像稿（可全部为生成素材）
W7答辩 deck、模拟评委追问、伦理/表述页定稿
W8排版、网页或 PDF 冻结、素材备份

3 工作分解（WBS）

3.1 系统与科学叙事（纸面 / 屏幕）

  • 将「皮质醇检测—显色—触觉」写成故事内自洽的因果链；可脚注「现实中需哪些条件」——无需验证。
  • 废热、气孔、渗透压：叙事级设定即可，图注或小字写明推测边界。
  • 光合微生物与复苏仪式：与世界观一致，可按科普级资料避免明显硬伤。
  • 输出：系统框图、时序故事板（焦虑暴发 / 高负载 / 长期弃置）。

3.2 设计资产

  • 四层叠层与 A–E 图例统一命名（与概念页同源更佳）。
  • 概念外观：尺度、握持区、气孔排布——重画面可读性，不重工艺可实现性。
  • 输出：Figma/AI 展板、HTML 概念页、可选动态分镜。

3.3 技术叙事支撑（书桌工作，非动手验证）

  1. 微流控/比色：公开研究中常见到什么程度——数句背景 + 参考链接即可。
  2. 水凝胶与渗透压：科普/综述级理解，支撑「质感变化」叙述。
  3. 温敏材料：作类比，说明叙事隐喻，不承诺配方。
  4. 生物电/MFC：若写入设定，用「极低功率示意」一笔带过并标虚构强度。

3.4 展示与传播

  • 项目介绍页、概念页、规划书三份材料术语一致。
  • 短片/幻灯结构：问题 → 机制 → 仪式 → 轮回（各段可短至 20–40 秒口播）。

3.5 合规与表述（贯穿全程）

  • 「感官勒索」等强措辞备学术版 / 大众版；对外保留思辨设计定位。
  • 本阶段无真实人体数据与生物样本；若将来扩展再补合规条款。

4 角色与协作（最小配置）

角色职责
项目负责人范围、叙事口径、截止日期。
叙事／视觉系统图、KV、分镜、网站与幻灯。
科学／设定顾问查资料、挑硬伤、不写实验方案。
伦理／表述「概念 vs 产品」措辞、答辩敏感问答。

一人团队：叙事 + 视觉 + 自编「我们不承诺什么」一节即可。

5 风险与应对

风险应对
被误读成「已能做成的产品」首页/封底固定概念提案；系统图旁注明推测与虚构强度。
评审质疑「操纵用户」与注意力经济对照；强调虚构场景与价值讨论，非销售承诺。
设定被指「不科学」脚注链到综述或科普；承认边界，诗学/伦理意图可优先于工程细节。
时间不足砍掉文献扩展，保住一条主叙事 + 一套主视觉。

6 伦理与表述备忘录

需在对外材料中可追溯；本阶段无实验与人体采样。

  • 不宣称医疗诊断或治疗；皮质醇等为设定元素，非实测指标。
  • 本阶段无生物实验与人体接触；若图像中出现体液隐喻，标注为象征性。
  • 「强制户外」「心理厌恶」等可 soften 为强提示、可逆反馈、仪式性叙事。
  • 若被问及「能否定做」：当前为叙事与视觉层面的概念作品，实物委托须另立范围。
